HTML BASIC TAGS

TAG DESCRITION
!DOCTYPE> All HTML documents must start with a !DOCTYPE> declaration.
HTML The html> tag is the container for all other HTML elements (except for the !DOCTYPE> tag).
HEAD The head> element is a container for metadata (data about data) and is placed between the html> tag and the body> tag.
TITLE The title> tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
BODY The element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
H1> TO H6> The h1> to h6> tags are used to define HTML headings.
P> The p> tag defines a paragraph.
BR> The br> tag inserts a single line break.
HR> The hr> tag defines a thematic break in an HTML page (e.g. a shift of topic).
!--...--> The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.
